Lightweight Web Animations

Why This Project?

Many developers rely on heavy animation libraries (like AOS - Animate On Scroll) that can add up to 330KB to your bundle size, even when you only need 1-2 animation effects. This can significantly impact your website's loading time and performance.

Our Solution

This lightweight, modular animation system:

  • Eliminates the need for heavy npm packages
  • Provides individual animation effects (fade, flip, zoom) that you can copy-paste as needed
  • Reduces bundle size by only including the animations you actually use
  • Maintains smooth performance with optimized CSS transitions
  • Works seamlessly with scroll-based triggers
